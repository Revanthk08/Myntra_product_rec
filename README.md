## Setup
<div align ="left">

  ##### Clone the repository
```bash
git clone https://github.com/Revanthk08/Myntra_product_rec.git
```
  ##### Move to the desired folder
```bash
cd ./(filename)
```
  ##### Download the dependencies
```bash
flutter pub get
```

  ##### To run the app, simply write
```bash
flutter run
```
